# Todo Pet App

Fullstack-приложение для управления списком задач (Todo).
Проект демонстрирует работу с REST API, CRUD-операциями, пользовательским React Hooks, фильтрацией и сортировкой данных.

---

## Возможности

* Просмотр списка задач, создание, редактирование, удаление задачи
* Фильтрация задач:
  * Все задачи
  * Выполненные
  * Не выполненные
* Сортировка задач:
  * Сначала выполненные
  * Сначала не выполненные
* Использование модального окна для создания и редактирования задач
* Обработка ошибок как на клиентской, так и на серверной стороне

---

# Технологии

## Backend

* Node.js
* Express
* REST API
* CRUD-операции
* uuidv4 для генерации уникальных идентификаторов
* JSON-файл в качестве виртуальной базы данных

## Frontend

* React
* JavaScript
* SCSS
* BEM
* React Hooks (`useState`, `useEffect`)
* Пользовательский Hook (`useTodos`)
* Fetch API
* Компонентный подход

# Запуск проекта

## 1. Клонировать репозиторий

```bash
git clone https://github.com/igor-drapeza/todo-pet-app.git
```

## 2. Установить зависимости

### Сервер

```bash
cd server
npm i
```

### Клиент

```bash
cd client
npm i
```

## 3. Запустить сервер

```bash
npm run dev
```

## 4. Запустить клиент

```bash
npm run dev
```
